# PRODIGY_DS_02

## Task 02: Data Cleaning and Exploratory Data Analysis (EDA)

### Objective
Perform data cleaning and exploratory data analysis on the Titanic dataset to identify patterns, trends, and relationships between variables.

### Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### Steps Performed
1. Loaded the Titanic dataset
2. Handled missing values
3. Removed unnecessary columns
4. Performed Exploratory Data Analysis (EDA)
5. Created visualizations
6. Generated insights from the data

### Key Insights
- Female passengers had a higher survival rate than male passengers.
- First-class passengers were more likely to survive.
- Most passengers were between 20 and 40 years old.
- Higher fare-paying passengers generally had better survival chances.

### Outcome
Successfully completed data cleaning and EDA on the Titanic dataset and identified meaningful patterns through visualization and statistical analysis.
